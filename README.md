# 2026CV-js

個人簡歷頁面，採用純 HTML、CSS、JavaScript 開發，無需任何框架。

## 功能特色

- **雙欄佈局設計**：左側顯示個人資訊、聯絡方式、關於我和技能；右側顯示教育經歷、里程碑和長期目標
- **互動式文字**：點擊描述文字可切換顏色（預設色 ↔ #004B97）
- **展開/收合功能**：點擊 "See All" 按鈕展開完整描述，點擊文字區塊可收合

## 使用方式

### 安裝依賴（可選）

如果需要使用開發伺服器：

```bash
yarn install
```

### 啟動開發伺服器

```bash
yarn start
```

或直接使用瀏覽器開啟 `index.html` 檔案。

## 互動功能說明

### Scenario 1: 顯示描述文字
- **Given**: 點擊描述文字
- **When**: 描述文字顏色會變更為 #004B97
- **Then**: 再點擊一次描述文字則恢復成預設值

### Scenario 2: 瀏覽完整描述
- **Given**: 想看到更多描述文字
- **When**: 點擊 "See All" 按鈕
- **Then**: 文字框向下展開，顯示完整描述內容

### Scenario 3: 收合描述文字
- **Given**: 收合描述文字
- **When**: 點擊描述文字區塊
- **Then**: 文字框向上收攏，恢復至預設狀態

## 技術規格

- 純前端實現，無需後端或資料庫
- 重新開啟頁面會回到預設狀態（無狀態儲存）
- 響應式設計，支援行動裝置
